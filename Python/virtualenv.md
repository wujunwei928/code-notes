
# 简介

# 安装
```bash
# ubuntu 安装
sudo apt-get install python-virtualenv

# easy_install 安装
sudo easy_install virtualenv

# pip 安装
pip install virtualenv
```

# 使用
```bash
# 创建一个名叫 venv 的虚拟环境
virtualenv venv

# 激活虚拟环境
# Linux，Mac
source venv/bin/activate
# Windows
venv\Scripts\activate

# 退出虚拟环境
deactivate
```
